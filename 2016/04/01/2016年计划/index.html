2016年计划

发表于 2016-04-01   |   分类于 学习   |     |   阅读次数

前言：开始学习前端有一阵子了，直到辞了实习去参加百度ife的课程才发现，原来自己被别人落下了那么多；一直鼓吹自己专攻前端，却被别人业余学习前端虐的体无完肤，再则人家压根就没想过入这一行，更是脸面无存。老黑博客写到“很多时候有人认为自己缺少的是平台，是经验，是别人的认可，是一个机会，或者其他云云，但你真正缺少的是未知惊喜之前的准备，没有准备好，也许惊喜就变成了惊讶”，原来我一直都没有准备好！

###16年整体的学习计划

  1. 学习高数（为css3的贝塞尔曲线以及矩阵开发做准备）
  2. 日常英语
  3. 技术部分
    1). webGL入门
    2). 前端可视化深入
    3). node.js掌握
    4). es7新特性学习
    5). Python以及go的入门
    6). reactjs以及Angularjs的掌握 backbone的深入
    7). react native app开发的掌握
    8). 数据结构与算法的加深学习
    9). 设计模式的学习
    10). http协议2.0的基本了解
  4. 书籍阅读（太泛，在此不一一列举）

差不多半年，试一下能不能搞得定，毕竟结果谁都说不定。尽心尽力而已！
